# Django Online Chat Room

Practice project for creating an online chat room using web socket
and deploying the web app at Heroku.

## Live web app:

[Link to heroku app](https://django-chat123.herokuapp.com/)

#### Tools used

- Python (django)
- HTML, CSS, JavaScript
- Bootstrap 5
- git, GitHub
- redis
- Channels
- Daphne
- Heroku
